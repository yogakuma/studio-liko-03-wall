FAQ

よくある質問

Q 予約方法を教えてください。 +

LINEまたは予約フォームから受け付けています。クラスごとに予約方法が異なる場合がございますので、各クラスのご案内ページをご確認ください。

Q 服装や持ち物はどうしたらいいですか？ +

動きやすい服装でお越しください。ヨガクラスはヨガマットの無料貸し出しがございます。お飲み物・タオル・ベビーサインクラスはお子さまのお気に入りのおもちゃがあると安心です。

Q お子さま連れで参加できますか？ +

ベビーサインクラス・マタニティ／産後ヨガ・キッズヨガ・オンラインヨガはお子さまとご一緒に参加いただけます。通常ヨガクラスは大人専用ですが、パーソナルレッスンならお子さまの同伴もご相談可能です。

Q 駐車場はありますか？ +

スタジオへのアクセス・駐車場のご案内は、本ページ下部のアクセスセクションをご確認ください。

Q 体験はどのクラスもありますか？ +

ヨガクラス・ベビーサインクラスともに体験レッスン／体験会をご用意しています。料金・持ち物などの詳細は、各クラスのご案内ページをご覧ください。

Q 発達がゆっくりな赤ちゃんでも参加できますか？ +

はい、ぜひお越しください。日本ベビーサイン協会認定マスター講師として、ダウン症など発達がゆっくりな赤ちゃんとそのご家族にも安心して通っていただけるよう、ご家族の状況に合わせて柔軟に対応しています。詳しくはLINEからお気軽にご相談ください。

ABOUT THE NAME

Liko

ハワイ語で「新芽・若葉・芽吹き・輝き」を意味する言葉。

子どもたちの名前の頭文字からとって生まれた、大切な名前です。つぼみだった子どもたちが光り輝いて、若々しい緑になっていきますようにと、願いを込めました。

小さなつぼみが若葉になり、やがて花を咲かせるように。親子の成長を、あたたかく見守る場所でありたいという想いが込められています。
