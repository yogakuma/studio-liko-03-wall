FAQ

よくある質問

Q シニアでも参加できますか？ +

はい。グループヨガは体力や関節の状態に合わせて進めます。難しいポーズや激しい動きはなく、必要に応じて椅子を使いながら参加できます。少人数制（最大7名）なので講師が一人ひとりの様子を見ながらご案内しますので、「久しぶりの運動でついていけるか不安」という方も安心してお越しください。50代・60代・70代の方にも多く通っていただいています。

Q ヨガが初めてでも参加できますか？ +

もちろんです。ポーズの名前を覚える必要はありません。呼吸のしかたから丁寧にご案内しますので、「何から始めていいか分からない」という方でも大丈夫です。初心者の方からのご参加が一番多いクラスです。

Q ホットヨガに通っていましたが、常温ヨガは違いますか？ +

大きく違います。常温のため無理に汗をかかず、体への負担が少ない分、呼吸やポーズの感覚に集中できます。「ホットが合わなかった」という方が通ってくださることも多いです。

Q 通常クラスに子ども連れで参加できますか？ +

グループヨガ・シニアヨガ・チャレンジヨガは、大人の方が自分の体と向き合うクラスのため、お子さま連れでの参加はご遠慮いただいています。赤ちゃんをお連れの方は、お子さま連れOKのマタニティ・産後ヨガクラスをご利用ください。

Q 産後いつから参加できますか？ +

体の回復状況によって異なります。まずはかかりつけ医にご確認いただいた上で、LINEでご相談ください。体の状態に合わせてご案内します。

Q どのクラスか分からないときは？ +

「迷っています」とLINEでご連絡ください。年齢・体の状態・目的をお聞きした上で、一緒に確認します。

Q 妊娠中でも通常クラスに参加できますか？ +

体調や妊娠週数によりご案内が変わるため、事前にご相談ください。通常クラスへの参加や産後ヨガへの参加など、状態に合わせてご案内します。

Q キッズヨガはどんな内容ですか？ +

遊びの要素を取り入れながら、楽しく体を動かすクラスです。姿勢や体幹づくりにもつながるよう、お子さまの成長に合わせて進めます。対象年齢やスケジュールの詳細はLINEでご確認ください。
